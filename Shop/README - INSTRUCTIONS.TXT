PROJECT TITLE: Simple Shop Project 
PURPOSE OF PROJECT: To create a functioning Shop with the ability to allow Customers to purchase Stock and allow the Shop to refill stock from a Supplier
VERSION or DATE: 09/10/2022
HOW TO START THIS PROJECT: Start by deciding the shop name and the price you want to sell your product. 
AUTHORS: Mazen Srari 
USER INSTRUCTIONS:
1 - create a Shop object with a price for your item. Shop will always start with 100 available stock and 1000 max stock. You'll be prompted by the program when the stock is running below a sustainable threshold.
2 - create a Customer object. Set a budget if necessary. If no budget is set, the budget will be considered unlimited
3 - create a Supplier from which the Shop can restock when running low in stock.
